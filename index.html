搭建图床

Posted on 2022-04-04

图床概述

图床：专门用于储存图片的服务器，将图片上传到服务器中后，可以在公网中通过指定的 URL 获取图片

优点：将图片的链接放到互联网上，可以在任何时间，任何地点、任何设备访问到图床上的图片

搭建准备

这里使用 GitHub 来搭建图床

使用 PicGo 上传图片

使用 jsDelivr 进行 CDN 加速

GitHub 搭建图床

  1. 新建一个仓库

    注意两点：

    ​ （1）该仓库公开（public）

    ​ （2）勾上Add a README file

  2. 创建仓库，这就搭建好了图床

PicGo 上传图片

使用传统的方法向 Github 上传图片太麻烦了，这里推荐使用一个开源图床工具 PicGo 来作为我们的图片上传工具

PicGo 的安装请见官网

不想写了，看看这篇文章，看完之后，你就应该会了

jsdelivr 进行 CDN 加速

上传到 Github 的图片有时候访问太慢了，这里使用 jsdelivr 进行 CDN 免费加速

设置的方法也很简单，只需要在我们 PicGo 图床配置中添加如下自定义域名即可；

https://cdn.jsdelivr.net/gh/用户名/仓库名

那么怎末访问图片呢

https://cdn.jsdelivr.net/gh/用户名/仓库名/文件路径

例如：https://cdn.jsdelivr.net/gh/lovesqiang/blogImage/banner3.jpg

当然，你还可以将你的仓库发布出去，加上你的版本号

例如：https://cdn.jsdelivr.net/gh/lovesqiang/cdn@1.3/images/bg/2.jpg

说明一下：这是两个不同的仓库，应该从 URL 路径看出来了吧

小知识

RGB就是红色R+绿色G+蓝色B，RGBA说得简单一点就是在RGB的基础上加进了一个通道Alpha.

语法: 以上R、G、B三个参数，正整数值的取值范围为：0 - 255。

R：红色值。正整数 | 百分数

G：绿色值。正整数 | 百分数

B：蓝色值。正整数| 百分数

A：透明度。取值0~1之间
